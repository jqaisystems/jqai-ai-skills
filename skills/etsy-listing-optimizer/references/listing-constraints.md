# Listing Constraints

Use these generic constraints when optimizing Etsy or similar marketplace listings. If the user gives current platform rules, follow the user's current rules.

## Title

- Keep the title readable for humans.
- Put the strongest buyer-intent phrase near the front.
- Avoid repeating the same phrase in slightly different forms.
- Do not use unrelated trends, competitor names, or trademarked terms unless the user owns or is allowed to use them.

## Tags

- Use relevant, distinct tags.
- Avoid wasting tag slots on near-duplicates when broader coverage is possible.
- Mix product type, use case, style, audience, occasion, and material when accurate.
- Do not include false attributes, restricted claims, or competitor brands.

## Description

Use this structure by default:

1. First line: clear product identity and main use case.
2. Short benefit paragraph.
3. Details: material, format, sizing, compatibility, download/use instructions, or production notes.
4. Gift/use cases when relevant.
5. Care, delivery, or policy notes only when accurate.

## Images And Attributes

Check whether the listing needs:

- Clear primary image.
- Detail or close-up image.
- Scale or sizing context.
- Variant image.
- Lifestyle/use image.
- Accurate category and attributes.

## Human Review

Before publishing, verify:

- Product facts are accurate.
- Keywords match the actual item.
- No trademark or competitor misuse.
- No misleading delivery, material, sizing, or compatibility claims.
- The title and description still sound natural.
