# v0.3.0

JQ AI Skills is now a validated public skill library.

This release turns the project from a packaged collection into a more maintainable open-source library: every push and pull request now runs automated checks for skill structure, OpenAI metadata, README asset links, installer behavior, and blocking public-safety findings.

## What's changed

- Added a GitHub Actions workflow for skill validation.
- Added `tools/validate_skills.py` to check every public skill folder.
- Added `tools/check_public_safety.py` to fail CI on blocking safety findings while allowing intentional review notes.
- Smoke-tested `install.ps1` in CI.
- Added an install demo GIF, MP4, poster image, and repeatable renderer.
- Kept the public library at 11 installable skills with categorized README navigation.
