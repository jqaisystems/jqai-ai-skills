# v0.6.3

JQ AI Skills now has an update guide for installed skills.

This release adds a practical path for refreshing one installed skill, updating a reviewed set of skills, testing the copy in a temporary target, reloading the tool, and checking the result.

The stable 2026 social preview is unchanged.

## What's changed

- Added `docs/guides/update-installed-skills.md`.
- Documented one-skill updates, all-skill updates, temporary-target checks, reload checks, and local edit handling.
- Linked the guide from the README, start page, quick reference, install page, install FAQ, catalog, and changelog.
- Updated the README release badge to `v0.6.3`.
- Kept the stable 2026 social preview unchanged.

## Why it matters

The repo now covers the full first-use loop: choose, install, verify, troubleshoot, and update.

Visitors can refresh installed skills without guessing whether to pull, reinstall, reload, or check the copied folder first.
