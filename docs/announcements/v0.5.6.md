# v0.5.6

JQ AI Skills now has a skill review checklist.

This release adds `docs/guides/skill-review-checklist.md` so visitors can evaluate a new, modified, or third-party skill folder before installing it.

The stable 2026 social preview is unchanged.

## What's changed

- Added `docs/guides/skill-review-checklist.md`.
- Covered source checks, folder shape, trigger scope, permissions, scripts, sensitive material, metadata, test targets, first prompts, and final install decisions.
- Linked the checklist from the README, `START_HERE.md`, `INSTALL.md`, `SECURITY.md`, and catalog.
- Updated `CHANGELOG.md` with the v0.5.6 entry.
- Updated the README release badge to `v0.5.6`.
- Kept the stable 2026 social preview unchanged.

## Why it matters

Installable skills are more trustworthy when review happens before installation.

The checklist gives visitors a public-safe way to decide whether a skill is `READY`, needs `REVIEW`, or should be blocked before it reaches a real workspace.
