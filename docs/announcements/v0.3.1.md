# v0.3.1

JQ AI Skills now includes `research-brief-curator`, a public-source research briefing skill inspired by the AI News Curator workflow.

This release is focused on turning saved public links, newsletters, docs, release notes, and research snippets into short, human-reviewed daily or weekly briefs without exposing private source lists, raw prompts, logs, local paths, account data, or client material.

## What's changed

- Added `skills/research-brief-curator/` as the 12th installable public skill.
- Added source-boundary guidance for public links, private research, review holds, and archive decisions.
- Added a reusable brief template for public-ready highlights, needs-review items, private research, archive notes, and manual checks.
- Added `scripts/build_brief_skeleton.py` to format structured JSON records into a Markdown research brief skeleton.
- Updated README navigation, quick-start examples, and skill selection guidance.
