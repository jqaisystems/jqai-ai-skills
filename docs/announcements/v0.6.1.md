# v0.6.1

JQ AI Skills now has a quick reference walkthrough.

This release adds a clone-to-first-prompt example for visitors who want to use `QUICK_REFERENCE.md` without guessing the order of commands, checks, and follow-up examples.

The stable 2026 social preview is unchanged.

## What's changed

- Added `docs/examples/quick-reference-walkthrough.md`.
- Showed the path from cloning the repo to listing skills, running a temporary install check, installing `github-safe-publisher`, reloading the tool, and using the first safe prompt.
- Linked the walkthrough from the README, start page, quick reference, examples index, catalog, and changelog.
- Updated the README release badge to `v0.6.1`.
- Kept the stable 2026 social preview unchanged.

## Why it matters

The repo already had the short command map. This release turns that map into a concrete first-use route.

Visitors can now inspect the fastest safe path before installing anything into their normal tool folder.
