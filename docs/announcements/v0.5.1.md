# v0.5.1

JQ AI Skills now has stronger example coverage for three practical workflow skills.

This release adds fake-data samples for release announcements, review-gated outreach pipelines, and marketplace listing optimization. The goal is to make the skills easier to evaluate before installing or adapting them.

The stable 2026 social preview is unchanged.

## What's changed

- Added `docs/examples/release-announcement-writer-sample.md`.
- Added `docs/examples/outreach-pipeline-designer-sample.md`.
- Added `docs/examples/etsy-listing-optimizer-sample.md`.
- Linked the new examples from the README and catalog.
- Updated `docs/skill-quality-matrix.md` for v0.5.1 example coverage.
- Updated `CHANGELOG.md` with the v0.5.1 entry.
- Updated the README release badge to `v0.5.1`.
- Kept the stable 2026 social preview unchanged.

## Why it matters

Examples make reusable skills easier to trust.

Visitors can now inspect what three more skills are expected to produce: a release announcement pack, a human-reviewed outreach pipeline, and a fictional marketplace listing audit. Each sample uses fake data and keeps human review visible.
