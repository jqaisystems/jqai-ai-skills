# v0.3.5

JQ AI Skills now has a clearer first-run path for visitors who want to choose and install the right skill without reading every folder first.

This release improves onboarding rather than adding a new skill. The goal is simple: help a new user start with the right workflow.

## What's changed

- Added `docs/guides/skill-selection.md` as a practical guide to choosing a first skill.
- Added starter sets for public GitHub publishing, content and research, and demo launch workflows.
- Updated the README Quick Start with recommended first installs.
- Expanded the README skill selector to include the legacy `demo-animation` skill.
- Updated the README release badge to `v0.3.5`.
