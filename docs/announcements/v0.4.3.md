# v0.4.3

JQ AI Skills now has a root `CHANGELOG.md` file.

This release makes the public release history easier to inspect from the top level of the repo. Visitors can now see how the library moved from first public packaging into a clearer onboarding path with Start Here, Install, Troubleshooting, and now a visible changelog.

The stable 2026 social preview is unchanged.

## What's changed

- Added `CHANGELOG.md` as the root release-history index.
- Linked `CHANGELOG.md` from the README and `START_HERE.md`.
- Updated the README release badge to `v0.4.3`.
- Kept the stable 2026 social preview unchanged.

## Why it matters

The repo now has a more complete public trust path:

```text
START_HERE -> INSTALL -> TROUBLESHOOTING -> CHANGELOG -> catalog and examples
```

Visitors can check what changed, when onboarding docs were added, and which public proof assets belong to each release without digging through Git tags first.
