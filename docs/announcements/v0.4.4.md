# v0.4.4

JQ AI Skills now has a root `RELEASE_CHECKLIST.md` file.

This release makes the publishing routine itself visible: edit, review diffs, validate skills, run public-safety scans, stage named files, watch GitHub Actions, tag the release, update profile and website surfaces, then verify the live pages.

The stable 2026 social preview is unchanged.

## What's changed

- Added `RELEASE_CHECKLIST.md` as the safe public release routine.
- Linked `RELEASE_CHECKLIST.md` from the README and `START_HERE.md`.
- Updated `CHANGELOG.md` with the v0.4.4 entry.
- Updated the README release badge to `v0.4.4`.
- Kept the stable 2026 social preview unchanged.

## Why it matters

The repo now shows both reusable AI skills and the release habit behind them.

That matters because a public skills library should be more than a folder of prompts. It should show how changes are checked, scanned, versioned, announced, and verified before the next cycle starts.
