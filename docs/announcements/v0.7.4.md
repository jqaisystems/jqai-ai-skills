# v0.7.4

JQ AI Skills now includes a demo animation v2 walkthrough sample.

This release keeps the stable 2026 social preview unchanged. The update focuses on the recommended responsive demo skill: `demo-animation-v2`.

## What Changed

- Added `docs/examples/demo-animation-v2-walkthrough-sample.md`.
- Used a fictional Signal Desk product brief to show the expected structure of a finished responsive walkthrough.
- Documented screens, captions, timeline steps, desktop checks, mobile checks, and public-safety checks.
- Linked the sample from the README, `START_HERE.md`, `QUICK_REFERENCE.md`, examples index, workflow bundles, public proof index, catalog, quality matrix, and changelog.
- Updated the quality matrix to `v0.7.4` and added the demo walkthrough sample to the visible coverage signals.
- Updated the README release badge to `v0.7.4`.

## Why It Matters

The repo already had a strong `demo-animation-v2` template asset. The next trust question is what a finished demo should look like after the skill runs.

The new sample closes that gap without exposing private client work. Visitors can inspect the desired pacing, screen sequence, captions, timeline controls, mobile review checks, and safety boundaries before generating a demo for their own project.
