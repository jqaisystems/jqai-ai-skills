# v0.3.6

JQ AI Skills now has a stronger public proof layer around the skill library.

This release does not add a new skill. It makes the existing catalog easier to understand, inspect, and share by adding a complete public catalog and switching the social preview to a stable 2026 banner that does not need to change on every release.

## What's changed

- Added `docs/catalog.md` as the complete public catalog for all 12 installable skills.
- Linked the catalog from the README and the skill selection guide.
- Added recommended bundles for public GitHub publishing, content and research, and demo launch workflows.
- Updated `assets/github-social-preview.png` and `assets/github-social-preview.jpg` to the final stable 2026 preview.
- Updated the README release badge to `v0.3.6`.

## Why it matters

Visitors can now understand the library at three levels:

- README for a quick overview.
- `docs/guides/skill-selection.md` for choosing the first skill.
- `docs/catalog.md` for the full public catalog, bundles, and safe install defaults.

The social preview now uses `updated 2026` instead of a release-specific version so it can stay stable across future patch releases.
