# v0.6.0

JQ AI Skills now has a quick reference page.

This release adds `QUICK_REFERENCE.md` as a short command and link map for visitors who want the fastest route through install commands, first-use checks, examples, and release helpers.

The stable 2026 social preview is unchanged.

## What's changed

- Added `QUICK_REFERENCE.md`.
- Collected first-choice links, install commands, temporary-target checks, first prompt, common skill prompts, public-safe publishing checks, and visitor links.
- Linked the quick reference from the README, start page, install page, catalog, and release checklist.
- Updated `CHANGELOG.md` with the v0.6.0 entry.
- Updated the README release badge to `v0.6.0`.
- Kept the stable 2026 social preview unchanged.

## Why it matters

The repo now has a single page for people who want to act quickly without losing the safety habit.

The quick reference keeps the core path visible: choose one skill, install it, verify it, test safely, and review public artifacts before release.
