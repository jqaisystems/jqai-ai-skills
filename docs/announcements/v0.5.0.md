# v0.5.0

JQ AI Skills now has an adoption layer.

This release adds the public-facing pieces that help visitors evaluate, request, and contribute to the library: a roadmap, support guidance, issue templates, a pull request checklist, and a skill quality matrix.

The stable 2026 social preview is unchanged.

## What's changed

- Added `ROADMAP.md` for the public-safe direction of the skill library.
- Added `SUPPORT.md` for install help, public issues, sensitive reports, and business contact routing.
- Added issue templates for skill requests, install or bug reports, and docs feedback.
- Added a pull request template with public-safety and validation checks.
- Added `docs/skill-quality-matrix.md` to compare maturity, examples, safety sensitivity, and install notes.
- Updated the README, `START_HERE.md`, and `CHANGELOG.md` for v0.5.0.
- Kept the stable 2026 social preview unchanged.

## Why it matters

The repo is now easier to adopt, not just browse.

Visitors can see where the project is going, understand how to ask for help, file cleaner public issues, review contribution expectations, and choose a skill based on maturity and safety sensitivity.
