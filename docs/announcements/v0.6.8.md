# v0.6.8

JQ AI Skills now has a first-skill candidate pack for visitors who want ready fake files after reading the first skill walkthrough.

This release keeps the stable 2026 social preview unchanged. The update focuses on the first practical test: giving visitors a small fictional README and docs folder they can point `github-safe-publisher` at without inventing their own sample.

## What Changed

- Added `docs/examples/first-skill-candidate/`.
- Added a fake `Example Review Queue` README, release note, and workflow summary.
- Linked the candidate pack from the README, `START_HERE.md`, `QUICK_REFERENCE.md`, examples index, first skill walkthrough, first-run sample, public proof index, visitor paths, evaluation checklist, first skill scorecard, skill selection guide, catalog, and changelog.
- Updated the README release badge to `v0.6.8`.

## Why It Matters

The repo already helps visitors evaluate the library, choose one safe first skill, and understand the expected first-run output.

The candidate pack closes the next small gap: it gives visitors public-safe files they can use immediately, so the first run tests the review workflow instead of their ability to invent a fake project.
