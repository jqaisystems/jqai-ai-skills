# v0.5.4

JQ AI Skills now has a public-safe skill request example.

This release adds `docs/examples/skill-request-example.md` to show how someone can propose a new skill without posting private project material.

The stable 2026 social preview is unchanged.

## What's changed

- Added `docs/examples/skill-request-example.md`.
- Linked the example from the README, `CONTRIBUTING.md`, `SUPPORT.md`, `ROADMAP.md`, catalog, and examples index.
- Clarified that skill request and pull request examples should use fictional, public, or approved material.
- Updated `CHANGELOG.md` with the v0.5.4 entry.
- Updated the README release badge to `v0.5.4`.
- Kept the stable 2026 social preview unchanged.

## Why it matters

Contribution paths are easier to trust when the safe shape is visible.

Visitors can now see how to describe a repeatable workflow, expected inputs, desired output, and safety boundaries without publishing real notes, client details, raw prompts, local paths, or private links.
