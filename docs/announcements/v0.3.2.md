# v0.3.2

JQ AI Skills now includes a public example artifact for `research-brief-curator`, showing how fake structured records become a short Markdown brief with public-ready, needs-review, and archive decisions.

This is a small documentation release focused on making the new research briefing skill easier to understand before installation.

## What's changed

- Added `docs/examples/research-brief-curator-sample-items.json` with fake source records.
- Added `docs/examples/research-brief-curator-sample.md`, generated with `skills/research-brief-curator/scripts/build_brief_skeleton.py`.
- Linked the sample brief from the README as an example artifact.
