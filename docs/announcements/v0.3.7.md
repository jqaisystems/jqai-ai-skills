# v0.3.7

JQ AI Skills now includes a dedicated workflow bundles example page for people who want to combine skills into practical public-safe workflows.

This release keeps the stable 2026 social preview unchanged. The update focuses on making the library easier to use after a visitor understands the individual skills.

## What's changed

- Added `docs/examples/workflow-bundles.md` with three reusable multi-skill paths:
  - Public GitHub publishing.
  - Content and research.
  - Demo and launch.
- Linked the workflow bundles page from the README, the complete catalog, and the skill selection guide.
- Updated the README release badge to `v0.3.7`.

## Why it matters

The repo now shows both single-skill entry points and practical skill combinations. A visitor can start with one skill, then move into a small bundle when the job needs safety review, public proof, release copy, source curation, or demo launch support.
