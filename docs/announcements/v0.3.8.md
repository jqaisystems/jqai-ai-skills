# v0.3.8

JQ AI Skills now includes a one-minute install guide for visitors who want to try one skill before exploring the full catalog.

This release keeps the stable 2026 social preview unchanged. The update focuses on reducing first-run friction: choose one skill, install it, reload your tool, and run a first prompt.

## What's changed

- Added `docs/guides/one-minute-install.md`.
- Linked the guide from the README, complete catalog, and skill selection guide.
- Updated the README release badge to `v0.3.8`.

## Why it matters

The repo already has a catalog, a selection guide, and workflow bundles. The new guide adds the missing first-run proof: a short path from public repo to installed local skill, with expected output and safe first-run rules.
