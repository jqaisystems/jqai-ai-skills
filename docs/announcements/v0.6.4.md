# v0.6.4

JQ AI Skills now has a public proof index.

This release adds one page that gathers the repo's install, safety, sample, system, and visual proof assets so visitors can evaluate the library quickly without hunting through the whole documentation tree.

The stable 2026 social preview is unchanged.

## What's changed

- Added `docs/examples/public-proof-index.md`.
- Grouped proof assets into fast review, install and first-use, public-safe publishing, skill output samples, system and library proof, and visual proof assets.
- Linked the proof index from the README, start page, quick reference, examples index, visitor paths, catalog, and changelog.
- Updated the README release badge to `v0.6.4`.
- Kept the stable 2026 social preview unchanged.

## Why it matters

The repo now has a single credibility entry point.

Visitors can inspect the public evidence behind the library before deciding whether to install a skill: setup checks, safety reviews, sample outputs, release discipline, and visual assets are all one click away.
