# v0.5.3

JQ AI Skills now has a skill anatomy guide.

This release adds `docs/guides/skill-anatomy.md` as a field guide for understanding what each required and optional file does inside an installable skill folder.

The stable 2026 social preview is unchanged.

## What's changed

- Added `docs/guides/skill-anatomy.md`.
- Documented the required `SKILL.md` and `agents/openai.yaml` files.
- Documented optional `references/`, `assets/`, and `scripts/` folders.
- Added install review and creation checklists.
- Linked the guide from the README, `START_HERE.md`, catalog, examples index, and skill quality matrix.
- Updated `docs/skill-quality-matrix.md` to `v0.5.3`.
- Updated `CHANGELOG.md` with the v0.5.3 entry.
- Updated the README release badge to `v0.5.3`.
- Kept the stable 2026 social preview unchanged.

## Why it matters

Visitors can now understand the folder contract before installing or adapting a skill.

That makes the library easier to inspect: what is required, what is optional, where longer guidance belongs, where scripts belong, and how to review a skill before use.
