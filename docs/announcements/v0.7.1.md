# v0.7.1

JQ AI Skills now includes a refreshed skill quality matrix.

This release keeps the stable 2026 social preview unchanged. The update focuses on keeping the public adoption layer honest after the first install proof work: the quality matrix now reflects the current examples, proof assets, safety sensitivity, and safest install notes across all public skills.

## What Changed

- Updated `docs/skill-quality-matrix.md` from `v0.5.3` to `v0.7.1`.
- Added current visible proof signals for the first install proof, first-skill candidate pack, expected review, public proof index, visitor paths, install verification, examples index, and catalog.
- Clarified public maturity, safety sensitivity, and safest install notes for all public skills.
- Added current gaps to watch so visitors can see where utility-skill proof can improve next.
- Linked the refreshed matrix from the README, `START_HERE.md`, `QUICK_REFERENCE.md`, catalog, and changelog.
- Updated the README release badge to `v0.7.1`.

## Why It Matters

The repo now has a strong first-install proof trail. The quality matrix should match that trust path, so visitors can quickly see which skills have the strongest public examples, which skills need the most care with private inputs, and which skills are safest to install first.

This is a credibility release: it closes a stale-version signal and makes the adoption layer easier to inspect.
