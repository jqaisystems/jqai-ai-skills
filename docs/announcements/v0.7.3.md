# v0.7.3

JQ AI Skills now includes a utility skill proof pack.

This release keeps the stable 2026 social preview unchanged. The update focuses on the smaller utility skills that are useful but previously had lighter proof coverage: `web-scraper`, `code-deduplicator`, and `em-dash-remover`.

## What Changed

- Added `docs/examples/utility-skill-proof-pack.md`.
- Added fake-data output shapes for public or authorized page extraction, small code deduplication, and copy cleanup.
- Linked the proof pack from the README, `START_HERE.md`, `QUICK_REFERENCE.md`, examples index, public proof index, catalog, quality matrix, and changelog.
- Updated the quality matrix to `v0.7.3` and added the utility proof pack to the visible coverage signals.
- Updated the README release badge to `v0.7.3`.

## Why It Matters

The repo already has strong proof for first install, update after release, safety review, and public-safe publishing workflows. This release adds concrete output proof for the smaller utilities so visitors can inspect what they do before installing them.

The goal is not to make lightweight skills feel heavy. The goal is to give each one enough public evidence to be understandable, testable, and safe to evaluate.

## Safety Note

The proof pack uses fictional pages, code, and copy. It does not include private source, account data, exports, browser profiles, raw logs, credentials, customer records, local machine paths, or unpublished implementation notes.
