# v0.3.3

JQ AI Skills now includes a public example artifact for `github-safe-publisher`, showing what a safe review output can look like before a public GitHub commit or push.

This release strengthens the safety and publishing side of the library by making the review gate more concrete: artifact type, files inspected, scanner summary, verdict examples, manual checks, and named-file staging.

## What's changed

- Added `docs/examples/github-safe-publisher-sample-review.md` with a fake public-safety review.
- Updated the README example artifacts section with the new safety review sample.
- Updated the README release badge to `v0.3.3`.
- Includes the stable launch media filenames already present on `main`.
