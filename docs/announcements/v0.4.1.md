# v0.4.1

JQ AI Skills now has a root `INSTALL.md` command reference.

This release makes the install path easier to find from the top level of the repo. `START_HERE.md` remains the first visitor route, while `INSTALL.md` gives direct commands for listing skills, installing one skill, installing all skills, using a custom test target, reloading the tool, and running the first prompt.

The stable 2026 social preview is unchanged.

## What's changed

- Added `INSTALL.md` as the root install command reference.
- Linked `INSTALL.md` from the README, `START_HERE.md`, catalog, and one-minute install guide.
- Updated the README release badge to `v0.4.1`.

## Why it matters

The repo now has clearer top-level navigation:

```text
START_HERE -> choose -> INSTALL -> run -> understand output -> use bundles or catalog
```

Visitors can now decide what to install from the onboarding page, then jump straight to the exact install command without hunting through the README.
