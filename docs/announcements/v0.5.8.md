# v0.5.8

JQ AI Skills now has an install smoke-test sample.

This release adds `docs/examples/install-smoke-test-sample.md` to show what a disposable temporary-target install check should look like before using a skill on real work.

The stable 2026 social preview is unchanged.

## What's changed

- Added `docs/examples/install-smoke-test-sample.md`.
- Showed expected Windows PowerShell and macOS/Linux checks for listing skills, copying one folder into `.tmp/skill-install-smoke`, confirming copied files, and cleaning up.
- Linked the sample from the README, examples index, install verification guide, install page, start page, troubleshooting guide, and catalog.
- Updated `CHANGELOG.md` with the v0.5.8 entry.
- Updated the README release badge to `v0.5.8`.
- Kept the stable 2026 social preview unchanged.

## Why it matters

The install verification guide is easier to trust when visitors can compare their own terminal output with a public-safe example.

The sample makes the first install more concrete: use a temporary target, confirm the whole folder copied, reload the tool, run a fake first prompt, and only then use the skill on real work.
