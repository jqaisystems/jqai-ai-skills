# v0.6.2

JQ AI Skills now has a visitor paths page.

This release adds a compact guide for people who want to evaluate the library, install one skill, or publish public proof safely without guessing which document comes next.

The stable 2026 social preview is unchanged.

## What's changed

- Added `docs/examples/visitor-paths.md`.
- Organized the main visitor flows into three routes: evaluate the library, install one skill, and publish public proof safely.
- Linked the visitor paths page from the README, start page, quick reference, examples index, catalog, and changelog.
- Updated the README release badge to `v0.6.2`.
- Kept the stable 2026 social preview unchanged.

## Why it matters

The repo has a growing set of guides and samples. This release gives visitors a simple routing layer so they can choose the right path before reading everything.
