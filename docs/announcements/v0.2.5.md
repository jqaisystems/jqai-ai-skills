# JQ AI Skills v0.2.5

JQ AI Skills now includes a faster local install path and a new release communication skill.

The `v0.2.5` release adds Windows and macOS/Linux installer scripts, updates the README around one-command skill installation, and introduces `release-announcement-writer` for turning changelogs, shipped features, and repo updates into GitHub release notes, website blurbs, and short launch posts.

This release keeps the library focused on repeatable AI workflows that can be installed, reviewed, improved, and shared.

Explore the release: https://github.com/jqaisystems/jqai-ai-skills/releases/tag/v0.2.5
