# v0.5.9

JQ AI Skills now has a skill install FAQ.

This release adds `docs/guides/install-faq.md` to answer common first-install questions before visitors need troubleshooting.

The stable 2026 social preview is unchanged.

## What's changed

- Added `docs/guides/install-faq.md`.
- Covered first skill choice, install-one versus install-all, copied folder shape, default and custom targets, install checks, reload issues, first prompts, safe first-test boundaries, updates, removal, and when to use `skill-reviewer`.
- Linked the FAQ from the README, start page, install page, one-minute install guide, install verification guide, troubleshooting guide, and catalog.
- Updated `CHANGELOG.md` with the v0.5.9 entry.
- Updated the README release badge to `v0.5.9`.
- Kept the stable 2026 social preview unchanged.

## Why it matters

Visitors now have a quick answer layer between install commands and full troubleshooting.

The FAQ makes the first-use path easier to follow: choose one skill, copy the whole folder, reload the tool, test with safe material, and update or remove skills intentionally.
