# v0.5.2

JQ AI Skills now has a public-safe examples index.

This release adds `docs/examples/README.md` as the landing page for first-run samples, fake-data examples, workflow bundles, and sample data. It gives visitors one place to inspect expected output before installing or adapting a skill.

The stable 2026 social preview is unchanged.

## What's changed

- Added `docs/examples/README.md`.
- Linked the examples index from the README, `START_HERE.md`, catalog, and skill quality matrix.
- Updated `docs/skill-quality-matrix.md` to `v0.5.2`.
- Updated `CHANGELOG.md` with the v0.5.2 entry.
- Updated the README release badge to `v0.5.2`.
- Kept the stable 2026 social preview unchanged.

## Why it matters

Examples are easier to trust when they are easy to find.

Visitors can now start from one index and choose between safety review samples, case study output, research brief output, release copy, outreach workflow design, marketplace listing optimization, workflow bundles, and sample data.
