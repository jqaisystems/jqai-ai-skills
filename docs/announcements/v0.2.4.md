# JQ AI Skills v0.2.4

JQ AI Skills is now easier to inspect, install, and share as a public skills library for Codex, Claude Code, and agentic developer workflows.

The `v0.2.4` release adds Codex-specific install instructions, a short release announcement, and clearer public presentation around the current skills library.

The library itself includes packaged skill folders, OpenAI metadata, a reusable skill template, contribution guidance, preview assets, and safety-reviewed workflows for publishing, scraping, skill review, copy cleanup, code deduplication, and demo generation.

Explore the release: https://github.com/jqaisystems/jqai-ai-skills/releases/tag/v0.2.4
