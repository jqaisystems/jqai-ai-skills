# v0.4.2

JQ AI Skills now has a root `TROUBLESHOOTING.md` page for install and reload issues.

This release gives visitors a clear place to go after installation if a skill does not appear, the shell command fails, the wrong target folder was used, or they need to reinstall or remove one skill safely.

The stable 2026 social preview is unchanged.

## What's changed

- Added `TROUBLESHOOTING.md` as the root install support page.
- Linked `TROUBLESHOOTING.md` from the README, `START_HERE.md`, `INSTALL.md`, catalog, and one-minute install guide.
- Updated the README release badge to `v0.4.2`.

## Why it matters

The repo now has a more complete first-use path:

```text
START_HERE -> choose -> INSTALL -> troubleshoot if needed -> run -> understand output -> use bundles or catalog
```

Visitors can now diagnose common setup issues without guessing whether the problem is reload, folder target, shell permissions, or skill naming.
