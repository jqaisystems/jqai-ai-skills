# v0.4.0

JQ AI Skills now has a top-level `START_HERE.md` onboarding path.

This release keeps the stable 2026 social preview unchanged. The update marks the public onboarding path as complete: visitors can understand the repo, choose one skill, install it, run a first prompt, compare the output, and then move into bundles or the full catalog.

## What's changed

- Added `START_HERE.md` as the top-level visitor route.
- Linked `START_HERE.md` from the README, complete catalog, and one-minute install guide.
- Updated the README release badge to `v0.4.0`.

## Why it matters

The repo now has one clear entry point for the full visitor journey:

```text
START_HERE -> choose -> install -> run -> understand output -> use bundles or catalog
```

That makes the library easier to inspect, share, and try without needing to understand every skill folder first.
