# v0.5.7

JQ AI Skills now has a skill reviewer sample.

This release adds `docs/examples/skill-reviewer-sample.md` to show how a review can classify fictional skill folders as `READY`, `REVIEW`, or `BLOCK` before installation.

The stable 2026 social preview is unchanged.

## What's changed

- Added `docs/examples/skill-reviewer-sample.md`.
- Showed one ready candidate, one review-needed candidate, and one blocked candidate using fake skill folders.
- Linked the sample from the README, examples index, catalog, skill review checklist, and `SECURITY.md`.
- Updated `CHANGELOG.md` with the v0.5.7 entry.
- Updated the README release badge to `v0.5.7`.
- Kept the stable 2026 social preview unchanged.

## Why it matters

The checklist is stronger when visitors can see it applied.

The sample makes the review habit concrete: inspect the folder, read scripts and metadata, flag risky instructions, and decide whether to test, revise, or reject before installing a skill.
